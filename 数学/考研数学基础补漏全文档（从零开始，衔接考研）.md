# 第一部分：必备基础运算（找回计算本能，考研计算不卡壳）

## 1.1 指数运算（核心，考研极限、导数常考）

### 1.1.1 基本定义

若a>0且a≠1，n为实数，则：aⁿ 表示n个a相乘（n为正整数）；n为负数时，a⁻ⁿ = 1/aⁿ；n为分数时，a^(m/n) = ⁿ√(aᵐ)（比如a^(1/2)=√a，a^(2/3)=³√(a²)）；n为0时，a⁰=1（a≠0，0⁰无意义，考研不考）。

### 1.1.2 必考运算公式（记死，不用推导）

- aᵐ × aⁿ = a^(m+n)（同底数幂相乘，指数相加）

- aᵐ ÷ aⁿ = a^(m-n)（同底数幂相除，指数相减，a≠0）

- (aᵐ)ⁿ = a^(mn)（幂的乘方，指数相乘）

- (ab)ⁿ = aⁿbⁿ（积的乘方，分别乘方）

- a⁰ = 1（任何非零数的0次方都是1，0⁰无意义）

- (a/b)ⁿ = aⁿ/bⁿ（商的乘方，分别乘方，b≠0，考研高频化简用到）

### 1.1.3 常用特例（考研高频）

2⁰=1，2¹=2，2²=4，2³=8，2⁴=16，2⁵=32，2⁶=64；10⁰=1，10¹=10，10²=100，10³=1000（对数会用到）；e⁰=1，e¹=e（e≈2.718，自然指数，考研高数高频出现）。

### 1.1.4 易错点提示（考研常踩坑）

1. 0的正指数幂为0，0的负指数幂无意义（比如0²=0，0⁻²无意义）；2. 负数的分数指数幂需注意定义域（比如(-2)^(1/2)无意义，(-2)^(2/3)=³√4，考研一般不考负数的分数指数幂，重点记正数情况）；3. a^(m+n)≠aᵐ + aⁿ（比如2²+2³=4+8=12≠2^(2+3)=32）。

## 1.2 对数运算（考研极限、导数、积分常考）

### 1.2.1 基本定义（和指数互为逆运算，记牢对应关系）

若a>0且a≠1，aˣ = N，则x = logₐN（读作“以a为底N的对数”）。其中，a叫对数的底数，N叫对数的真数，真数N必须大于0（对数的定义域，考研求定义域常考）。

关键对应：aˣ = N ↔ logₐN = x（比如2³=8 ↔ log₂8=3；e²≈7.389 ↔ ln7.389≈2）。

### 1.2.2 必考运算公式（重点记前4个，考研高频，核心公式附推导）

- logₐ1 = 0（推导：由指数定义，a⁰=1，结合对数与指数的逆关系，logₐ1=0；任何底数，真数为1，对数为0）

- logₐa = 1（推导：由指数定义，a¹=a，结合逆关系，logₐa=1；底数和真数相等，对数为1）

- logₐ(MN) = logₐM + logₐN（推导：设logₐM = x，logₐN = y，则aˣ=M，aʸ=N，因此MN=aˣ×aʸ=a^(x+y)，再由逆关系，logₐ(MN)=x+y=logₐM + logₐN；积的对数=对数的和，M>0，N>0）

- logₐ(M/N) = logₐM - logₐN（推导：同理，设logₐM=x，logₐN=y，则M=aˣ，N=aʸ，M/N=aˣ÷aʸ=a^(x-y)，故logₐ(M/N)=x-y=logₐM - logₐN；商的对数=对数的差，M>0，N>0）

- logₐ(Mⁿ) = nlogₐM（推导：设logₐM=x，则M=aˣ，Mⁿ=(aˣ)ⁿ=a^(nx)，由逆关系，logₐ(Mⁿ)=nx=nlogₐM；幂的对数=指数×对数，M>0）

- 换底公式：logₐN = log_bN / log_ba（推导：设logₐN=x，log_bN=y，log_ba=z，则aˣ=N，bʸ=N，bᶻ=a，因此bʸ=N=(bᶻ)ˣ=b^(zx)，故y=zx，即x=y/z，也就是logₐN=log_bN / log_ba；b>0且b≠1，考研常用：logₐN = lnN / lna，ln是自然对数，底数为e≈2.718）

- 对数恒等式：a^(logₐN) = N（推导：设logₐN=x，则由对数定义，aˣ=N，代入左边得aˣ=N，等式成立；M>0，考研极限化简偶尔用到）

### 1.2.3 常用特例（考研高频）

ln1=0，lne=1，ln(eˣ)=x，e^(lnx)=x（x>0）；lg10=1，lg1=0，lg10ˣ=x（lg是常用对数，底数为10）；log₂2=1，log₂4=2，log₂8=3，log₂16=4（考研极限常考）。

### 1.2.4 易错点提示（考研常踩坑）

1. 真数必须大于0（比如log₂(-2)无意义，考研求对数函数定义域必考）；2. logₐ(M±N)≠logₐM±logₐN（比如log₂(4+8)=log₂12≠log₂4+log₂8=2+3=5）；3. ln x是自然对数（底数e），lg x是常用对数（底数10），二者不能混淆，考研中ln x使用频率远高于lg x。

## 1.3 代数基础运算（考研化简、解方程必备）

### 1.3.1 因式分解（核心，考研求极限、约分、求导常用，核心公式附推导）

- 平方差公式：a² - b² = (a + b)(a - b)（推导：右边展开=(a+b)(a-b)=a² -ab +ab -b²=a² -b²，与左边相等；必考，比如x² - 1=(x+1)(x-1)，x² - 4=(x+2)(x-2)）

- 完全平方公式：(a±b)² = a² ± 2ab + b²（推导：(a+b)²=(a+b)(a+b)=a² +ab +ab +b²=a²+2ab+b²；(a-b)²=(a-b)(a-b)=a² -ab -ab +b²=a²-2ab+b²；必考，比如x² + 2x + 1=(x+1)²，x² - 4x + 4=(x-2)²）

- 完全立方公式（简单了解，偶尔考）：(a±b)³ = a³ ± 3a²b + 3ab² ± b³

- 立方差/和公式（考研偶尔考）：a³ - b³ = (a - b)(a² + ab + b²)；a³ + b³ = (a + b)(a² - ab + b²)（推导：以立方差为例，右边展开=(a-b)(a²+ab+b²)=a³+a²b+ab² -a²b -ab² -b³=a³ -b³，与左边相等）

- 十字相乘法（简单版）：x² + (m+n)x + mn = (x+m)(x+n)（推导：右边展开=(x+m)(x+n)=x² +nx +mx +mn=x²+(m+n)x+mn，与左边相等；比如x² + 3x + 2=(x+1)(x+2)，x² - 5x + 6=(x-2)(x-3)）

- 提取公因式（基础，化简必用）：ax + ay = a(x + y)，x³ - x² = x²(x - 1)

### 1.3.2 一元二次方程（偶尔考，基础必备，衔接考研求根、判别式，求根公式附推导）

方程：ax² + bx + c = 0（a≠0），求根公式：x = [-b ± √(b² - 4ac)] / 2a（推导：用配方法，ax²+bx+c=0，两边除以a得x² + (b/a)x + c/a=0，配方得x² + (b/a)x + (b/(2a))² = (b/(2a))² - c/a，左边化为(x + b/(2a))²，右边通分=(b² - 4ac)/(4a²)，两边开方得x + b/(2a)=±√(b²-4ac)/(2a)，移项得x=[-b±√(b²-4ac)]/(2a)；考研求极限、求定义域偶尔用到）

判别式：Δ = b² - 4ac，Δ>0有两个不同实根，Δ=0有一个实根（二重根），Δ<0无实根（有两个共轭虚根，考研高数不考虚根，只需判断有无实根即可）。

根与系数关系（韦达定理，偶尔考）：x₁ + x₂ = -b/a，x₁x₂ = c/a（推导：由求根公式，x₁=[-b+√(b²-4ac)]/(2a)，x₂=[-b-√(b²-4ac)]/(2a)，两式相加得(x₁+x₂)=(-2b)/(2a)=-b/a；两式相乘得x₁x₂=[b² - (b²-4ac)]/(4a²)=4ac/(4a²)=c/a；比如x² - 3x + 2=0，根为1和2，1+2=3=-(-3)/1，1×2=2=2/1）。

### 1.3.3 不等式基础（考研求定义域、估值、单调性判断常用）

- 基本性质：a>b，c>0 → ac>bc；a>b，c<0 → ac<bc（不等号方向改变，考研解不等式常踩坑）；a>b，b>c → a>c；a>b → a±c>b±c。

- 常见不等式（必考）：x² ≥ 0（任何实数的平方非负）；|x| ≥ 0（绝对值非负）；a² + b² ≥ 2ab（推导：由(x-y)²≥0，令x=a，y=b，得(a-b)²≥0，展开得a²-2ab+b²≥0，移项得a²+b²≥2ab；基本不等式，考研估值、证明偶尔考）；√(ab) ≤ (a+b)/2（均值不等式，a>0，b>0，简单了解即可）。

- 简单绝对值不等式（必考，求定义域常用）：|x| < a（a>0）→ -a < x < a；|x| > a（a>0）→ x < -a 或 x > a；|ax + b| < c（c>0）→ -c < ax + b < c（比如|2x - 1| < 3 → -3 < 2x - 1 < 3 → -1 < x < 2）。

- 一元二次不等式（必考，求定义域常用）：ax² + bx + c > 0（a≠0），先判断Δ，再结合二次函数图像判断解集（比如x² - 3x + 2 > 0，因式分解为(x-1)(x-2)>0，解集为x<1或x>2）。

### 1.3.4 易错点提示（考研常踩坑）

1. 解不等式时，两边同乘/除以负数，不等号方向必须改变（比如-2x > 4 → x < -2，不是x > -2）；2. 一元二次不等式中，a的正负决定二次函数开口方向，影响解集（a>0开口向上，a<0开口向下）；3. 基本不等式a² + b² ≥ 2ab，等号成立条件是a=b，考研证明题偶尔用到。

# 第二部分：三角函数（考研高数核心基础，必补！）

## 2.1 单位圆与sinx、cosx的定义（重中之重，记死！）

### 2.1.1 单位圆定义

以原点(0,0)为圆心，半径r=1的圆，叫单位圆；从x轴正方向（右边）逆时针转角度x（角度可正可负，逆时针为正，顺时针为负），落到圆上一点P，点P的坐标为(cosx, sinx)（x可以是角度制，也可以是弧度制，考研中一律用弧度制，后续会补充弧度制基础）。

### 2.1.2 核心结论（一辈子不忘，考研高频，核心结论附逻辑推导）

- cosx = 点P的横坐标（左右位置）（逻辑：单位圆中，点P的横坐标对应x轴方向的投影，符合三角函数的定义，也是后续平方关系推导的基础）

- sinx = 点P的纵坐标（上下位置）（逻辑：同理，点P的纵坐标对应y轴方向的投影，与横坐标共同构成直角三角形的两条直角边）

- sinx和cosx的值，永远在[-1, 1]之间（推导：单位圆半径r=1，点P在圆上，横坐标和纵坐标的取值范围均为[-1,1]，因此sinx、cosx的值也在[-1,1]之间；因为单位圆半径为1，横坐标、纵坐标最大1，最小-1，考研判断函数值域常用）

- 若点P在单位圆上，则cos²x + sin²x = 1（平方关系，推导：由勾股定理，直角三角形OMP中，OM² + PM² = OP²，OM=cosx，PM=sinx，OP=1（半径），因此cos²x + sin²x = 1²=1；直接由勾股定理推导，永远成立，必考）

### 2.1.3 直角三角形辅助理解（初中定义，辅助记忆）

从点P向x轴作垂线，形成直角三角形OMP（O是原点，M是垂足）：

- 斜边OP=1（单位圆半径）

- 邻边OM = 横坐标 = cosx（挨着角度x的边）

- 对边PM = 纵坐标 = sinx（对着角度x的边）

- 初中定义：cosx = 邻边/斜边 = OM/OP = cosx/1 = cosx；sinx = 对边/斜边 = PM/OP = sinx/1 = sinx（和单位圆定义一致，帮助记忆）

### 2.1.4 弧度制基础（考研必备，重中之重，核心关系附推导）

考研数学中，三角函数的角度一律用弧度制，不用角度制，必须掌握弧度与角度的转换：

- 180° = π 弧度（π≈3.1416，推导：弧度制的定义是“弧长与半径的比值”，单位圆周长为2πr=2π（r=1），对应360°，因此半圆弧长为π，对应180°，即180°=π弧度；记死这个对应关系）

- 转换公式：弧度 = 角度 × (π/180°)；角度 = 弧度 × (180°/π)（推导：由180°=π弧度，可得1°=π/180°弧度，1弧度=180°/π，因此角度转弧度需乘π/180°，弧度转角度需乘180°/π）

- 常用角度与弧度转换（必考，背熟）：0°=0弧度，30°=π/6，45°=π/4，60°=π/3，90°=π/2，180°=π，270°=3π/2，360°=2π

- 弧长公式（简单了解）：l = rθ（r为半径，θ为弧度，推导：由弧度制定义，θ=l/r，变形得l=rθ；单位圆中r=1，弧长l=θ）

## 2.3 三角函数核心公式（考研高频，只记常用的，不记冷门的，重点记变形，核心公式附推导）

### 2.3.1 平方关系（必考，求极限、化简、求导常用）

sin²x + cos²x = 1（由单位圆半径为1推导，推导：直角三角形OMP中，OM=cosx，PM=sinx，OP=1，由勾股定理OM²+PM²=OP²，即cos²x+sin²x=1；永远成立）

变形（高频，必须会）：1 - sin²x = cos²x；1 - cos²x = sin²x（推导：由sin²x + cos²x = 1，移项即可得到；考研求极限约分、三角代换必考，比如(1 - cosx)/x²，需用1 - cos²x变形）

补充：1 + tan²x = sec²x；1 + cot²x = csc²x（简单了解，考研考得极少，可暂不记）

### 2.3.2 二倍角公式（考研求导、积分、三角代换必考，重点记，核心公式附推导）

- sin2x = 2sinxcosx（推导：由两角和公式sin(A+B)=sinAcosB+cosAsinB，令A=B=x，得sin(x+x)=sinxcosx+cosxsinx=2sinxcosx；重点，记牢，考研求导、积分高频）

- cos2x（3种形式，记前2种，变形必考）：
        

    - cos2x = cos²x - sin²x（推导：由两角和公式cos(A+B)=cosAcosB - sinAsinB，令A=B=x，得cos(x+x)=cosxcosx - sinxsinx=cos²x - sin²x；基础形式）

    - cos2x = 2cos²x - 1（推导：由平方关系sin²x=1 - cos²x，代入上式，cos2x=cos²x - (1 - cos²x)=cos²x -1 + cos²x=2cos²x -1；常用变形：cos²x = (1 + cos2x)/2，考研积分降幂必考）

    - cos2x = 1 - 2sin²x（推导：同理，由cos²x=1 - sin²x，代入cos2x=cos²x - sin²x，得cos2x=(1 - sin²x) - sin²x=1 - 2sin²x；常用变形：sin²x = (1 - cos2x)/2，考研积分降幂必考）

- tan2x = 2tanx/(1 - tan²x)（简单了解，考研考得少，推导：tan2x=sin2x/cos2x=2sinxcosx/(cos²x - sin²x)，分子分母同除以cos²x，得2tanx/(1 - tan²x)）

### 2.3.3 诱导公式（简单记，不用死背所有，记规律，考研偶尔用到，规律逻辑说明）

核心规律（奇变偶不变，符号看象限，记牢这个规律，所有诱导公式都能推导，规律逻辑说明）：

- “奇变偶不变”：诱导公式中，角度加上/减去kπ/2（k为整数），k为奇数时，sin↔cos，cos↔sin；k为偶数时，函数名不变（逻辑：kπ/2是90°的整数倍，每加/减90°，三角函数的定义会切换横坐标和纵坐标，即sin和cos互换，k为偶数时，角度旋转180°、360°等，函数名不变）。

- “符号看象限”：假设x是锐角，判断原函数在对应象限的正负，即为诱导公式的符号（逻辑：锐角x加上/减去kπ/2后，会落在不同象限，而三角函数在不同象限的正负是固定的，据此确定公式的符号）。

常用诱导公式（考研高频，直接记）：

- sin(π - x) = sinx；cos(π - x) = -cosx（推导：π - x=2×(π/2) - x，k=2（偶数），函数名不变；x为锐角，π - x在第二象限，sin在第二象限为正，cos在第二象限为负，故符号分别为正、负）

- sin(π + x) = -sinx；cos(π + x) = -cosx（推导：π + x=2×(π/2) + x，k=2（偶数），函数名不变；x为锐角，π + x在第三象限，sin、cos在第三象限均为负，故符号为负）

- sin(2π - x) = -sinx；cos(2π - x) = cosx（推导：2π - x=4×(π/2) - x，k=4（偶数），函数名不变；x为锐角，2π - x在第四象限，sin在第四象限为负，cos在第四象限为正，故符号分别为负、正）

- sin(π/2 - x) = cosx；cos(π/2 - x) = sinx（推导：π/2 - x=1×(π/2) - x，k=1（奇数），函数名互换；x为锐角，π/2 - x在第一象限，sin、cos在第一象限均为正，故符号为正）

- sin(-x) = -sinx（奇函数）；cos(-x) = cosx（偶函数，考研判断函数奇偶性常用）（推导：-x可看作0×(π/2) - x，k=0（偶数），函数名不变；-x在第四象限，sin在第四象限为负，cos在第四象限为正，故符号分别为负、正）

### 2.3.4 正切函数（辅助，考研考得少，简单了解，衔接求导）

tanx = sinx / cosx（cosx≠0，即x≠π/2 + kπ，k为整数，定义域考研偶尔考）

特殊角（简单记）：tan0=0，tan(π/6)=√3/3，tan(π/4)=1，tan(π/3)=√3，tanπ=0

常用公式：tan(π - x) = -tanx；tan(-x) = -tanx（奇函数）（推导：tan(π - x)=sin(π - x)/cos(π - x)=sinx/(-cosx)=-tanx；tan(-x)=sin(-x)/cos(-x)=(-sinx)/cosx=-tanx）

### 2.3.5 三角函数图像与性质（了解即可，帮助理解周期、单调性，衔接考研）

- sinx：周期2π，定义域R，值域[-1,1]，奇函数，在[0, π/2]上单调递增，在[π/2, 3π/2]上单调递减（考研判断单调性、值域常用）

- cosx：周期2π，定义域R，值域[-1,1]，偶函数，在[0, π]上单调递减，在[π, 2π]上单调递增

- tanx：周期π，定义域{x | x≠π/2 + kπ, k∈Z}，值域R，奇函数，在(-π/2, π/2)上单调递增

### 2.3.6 易错点提示（考研常踩坑）

1. 考研全程用弧度制，不要用角度制（比如求sin(π/2)，答案是1，不是sin90°=1，书写时必须用弧度制）；2. 二倍角公式的变形（cos²x、sin²x的降幂公式）是考研积分的重点，必须记牢；3. 诱导公式的符号容易错，记住“符号看象限”的规律，避免出错；4. tanx的定义域容易忽略，cosx≠0，即x≠π/2 + kπ。

# 第三部分：其他必备基础（衔接考研，简单补，不遗漏考点）

## 3.1 数列基础（考研高数、线代偶尔考，记核心，不深钻，核心公式附推导）

### 3.1.1 等差数列（简单了解，考研高数偶尔考）

定义：从第二项起，每一项与它的前一项的差等于同一个常数（公差d）。

通项公式：aₙ = a₁ + (n-1)d（a₁是首项，d是公差，n是项数）（推导：由定义，a₂=a₁+d，a₃=a₂+d=a₁+2d，a₄=a₃+d=a₁+3d，以此类推，第n项aₙ=a₁+(n-1)d）

前n项和：Sₙ = n(a₁ + aₙ)/2 = na₁ + n(n-1)d/2（考研偶尔考，记公式即可，不用推导，推导：倒序相加法，Sₙ=a₁+a₂+...+aₙ，Sₙ=aₙ+aₙ₋₁+...+a₁，两式相加得2Sₙ=n(a₁+aₙ)，故Sₙ=n(a₁+aₙ)/2；代入通项公式aₙ=a₁+(n-1)d，得Sₙ=na₁ + n(n-1)d/2）

常用性质：aₙ = aₘ + (n - m)d；若m + n = p + q，则aₘ + aₙ = aₚ + a_q（简单了解）

### 3.1.2 等比数列（简单了解，考研极限、高数偶尔考）

定义：从第二项起，每一项与它的前一项的比等于同一个常数（公比q），q≠0。

通项公式：aₙ = a₁q^(n-1)（a₁是首项，q是公比，q≠0，n是项数）（推导：由定义，a₂=a₁q，a₃=a₂q=a₁q²，a₄=a₃q=a₁q³，以此类推，第n项aₙ=a₁q^(n-1)）

前n项和：Sₙ = a₁(1 - qⁿ)/(1 - q)（q≠1）；Sₙ = na₁（q=1）（考研极限中偶尔用到，比如等比数列求和的极限，推导：错位相减法，Sₙ=a₁+a₁q+a₁q²+...+a₁q^(n-1)，qSₙ=a₁q+a₁q²+...+a₁qⁿ，两式相减得(1-q)Sₙ=a₁(1-qⁿ)，故Sₙ=a₁(1-qⁿ)/(1-q)；q=1时，所有项均为a₁，故Sₙ=na₁）

常用性质：aₙ = aₘq^(n - m)；若m + n = p + q，则aₘaₙ = aₚa_q（简单了解）

### 3.1.3 易错点提示

1. 等比数列公比q≠0，且q=1时，前n项和公式不同，避免混淆；2. 等差数列公差d可以为0（常数列），等比数列公比q可以为1（常数列）。

## 3.2 向量基础（衔接线代，简单补，线代会详细学，这里记基础）

### 3.2.1 平面向量（基础，衔接线代二维向量）

定义：既有大小又有方向的量，坐标表示为(x, y)（起点在原点）。

核心运算（考研线代基础，记牢）：

- 加法：两个向量(x₁,y₁)和(x₂,y₂)，和为(x₁+x₂, y₁+y₂)（平行四边形法则/三角形法则，逻辑：向量加法遵循“首尾相接”或“对角相加”，坐标对应相加，符合几何意义）

- 数乘：k(x₁,y₁)=(kx₁, ky₁)（k为实数，伸长/缩短向量，改变方向，逻辑：k>0时，向量方向不变，长度变为原来的k倍；k<0时，向量方向相反，长度变为原来的|k|倍）

- 数量积（点积，考研线代必考）：(x₁,y₁)·(x₂,y₂) = x₁x₂ + y₁y₂（结果是一个数，不是向量，逻辑：数量积是一个向量在另一个向量上的投影与另一个向量长度的乘积，坐标运算对应横坐标相乘加纵坐标相乘）

- 向量垂直：两个向量数量积为0，即x₁x₂ + y₁y₂ = 0（考研线代判断垂直常用，逻辑：垂直时，一个向量在另一个向量上的投影为0，故数量积为0）

### 3.2.2 空间向量（简单了解，衔接线代三维向量）

坐标表示为(x, y, z)，数量积：(x₁,y₁,z₁)·(x₂,y₂,z₂) = x₁x₂ + y₁y₂ + z₁z₂（线代会详细学，这里记基础即可）

## 3.3 绝对值与分段函数（考研高数第一章常考，重中之重）

### 3.3.1 绝对值定义（必考）

|x| = x（x≥0）；|x| = -x（x<0）（比如|2|=2，|-2|=2，|0|=0）

常用性质（考研化简常用）：|ab|=|a||b|；|a/b|=|a|/|b|（b≠0）；|a|≥0；|a|=| -a |

### 3.3.2 分段函数（必考，考研极限、导数常考）

定义：在不同的定义域区间，有不同的表达式的函数（比如绝对值函数就是分段函数）。

常见例子：

- f(x) = {x, x≥0；-x, x<0}（即f(x)=|x|，考研求极限、导数常考）

- f(x) = {x², x<1；2x - 1, x≥1}（考研判断分段函数连续性、求分段点导数必考）

核心要点：记清“分段点”（定义域的分界点，比如x=0、x=1），考研重点考分段点的极限、连续性和导数。

### 3.3.3 易错点提示

1. 求分段函数的极限、导数时，必须分左右极限、左右导数（分段点处），不能直接用一个表达式计算；2. 绝对值函数的导数在分段点处需要单独计算（比如f(x)=|x|，在x=0处不可导，考研常考）。

## 3.4 函数基础（补充，衔接考研高数第一章，之前遗漏）

### 3.4.1 函数的定义与定义域（考研高数必考）

函数：y = f(x)，x是自变量，y是因变量，定义域是x的取值范围（考研求定义域常考：对数真数>0、根号下≥0、分母≠0、三角函数定义域等）。

### 3.4.2 函数的奇偶性（考研高数常考，判断函数性质，核心定义逻辑）

- 奇函数：f(-x) = -f(x)（图像关于原点对称，比如sinx、tanx，逻辑：对于原点对称的点，函数值互为相反数）

- 偶函数：f(-x) = f(x)（图像关于y轴对称，比如cosx、|x|，逻辑：对于y轴对称的点，函数值相等）

- 易错点：判断奇偶性的前提是定义域关于原点对称（比如f(x)=x²，x∈[0,1]，定义域不关于原点对称，非奇非偶）

### 3.4.3 函数的单调性（简单了解，衔接考研导数判断单调性）

增函数：x₁ < x₂ → f(x₁) < f(x₂)；减函数：x₁ < x₂ → f(x₁) > f(x₂)（考研后续用导数判断，这里记基础定义）
