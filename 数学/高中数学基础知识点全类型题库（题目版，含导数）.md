### （四）综合提升（12道，浅→深）

39. 化简：$\frac{\frac{1}{x} - \frac{1}{y}}{x - y}$=$\frac{\frac{y-x}{xy}}{x-y}$ = $-\frac{x-y}{xy} \times \frac{1}{x-y} = \frac{1}{xy}$

40. 化简：$\left(x + \frac{1}{x}\right)^2 - \left(x - \frac{1}{x}\right)^2$ = $x^2 + 2 + \frac{1}{x^2} - (x^2-2 + \frac{1}{x^2}) = 4  且 x\ne 0$

41. 已知 $x + y = 5$，$xy = 6$，求 $x^2 + y^2$ 
 - 解 $（x+y）^2 = x^2+2xy+y^2 = 25 $
 - 由  $2xy = 6 \times 2 = 12$
 - 因此 $x^2+y^2 = 25-12 = 13$
     
42. 已知 $a - b = 2$，$ab = 3$，求 $a^2 + b^2$
- 解答 已知 $a-b=2$，则$(a-b)^2 = a^2-2ab+b^2 = 4$
- 有 $ab=3$,则$2ab=6$,则有$(a-b)^2 = a^2-2ab+b^2 = a^2-6+b^2=4$ 
- 因此 $a^2 + b^2 = 4+6 = 10$

43. 已知 $x + \frac{1}{x} = 3$，求 $x^2 + \frac{1}{x^2}$
- 解答 $x+\frac{1}{x} = 3$,则有 $(x+\frac{1}{x})^2 = x^2 + 2 + \frac{1}{x^2} = 3^2$
- 因此 $x^2+\frac{1}{x^2} = 3^2-2 = 7$
