### （四）综合提升（12道，浅→深）

39. 化简：$\frac{\frac{1}{x} - \frac{1}{y}}{x - y}$=$\frac{\frac{y-x}{xy}}{x-y}$ = $-\frac{x-y}{xy} \times \frac{1}{x-y} = \frac{1}{xy}$

40. 化简：$\left(x + \frac{1}{x}\right)^2 - \left(x - \frac{1}{x}\right)^2$ = $x^2 + 2 + \frac{1}{x^2} - (x^2-2 + \frac{1}{x^2}) = 4  且 x\ne 0$

41. 已知 $x + y = 5$，$xy = 6$，求 $x^2 + y^2$ 
 - 解 $（x+y）^2 = x^2+2xy+y^2 = 25 $
 - 由  $2xy = 6 \times 2 = 12$
 - 因此 $x^2+y^2 = 25-12 = 13$
     
42. 已知 $a - b = 2$，$ab = 3$，求 $a^2 + b^2$
- 解答 已知 $a-b=2$，则$(a-b)^2 = a^2-2ab+b^2 = 4$
- 有 $ab=3$,则$2ab=6$,则有$(a-b)^2 = a^2-2ab+b^2 = a^2-6+b^2=4$ 
- 因此 $a^2 + b^2 = 4+6 = 10$

43. 已知 $x + \frac{1}{x} = 3$，求 $x^2 + \frac{1}{x^2}$
- 解答 $x+\frac{1}{x} = 3$,则有 $(x+\frac{1}{x})^2 = x^2 + 2 + \frac{1}{x^2} = 3^2$
- 因此 $x^2+\frac{1}{x^2} = 3^2-2 = 7$

44. 已知 $x - \frac{1}{x} = 2$，求 $x^2 + \frac{1}{x^2}$
- 解答 已知 $x-\frac{1}{x} = 2$,则有 $(x-\frac{1}{x})^2 = x^2 - 2+\frac{1}{x^2} = 2^2$
- 所以 $x^2+\frac{1}{x^2} = 2^2+2 = 6$ 且$x\ne0$,

45. 若 $x + y = 4$，$x^2 + y^2 = 10$，求 $xy$
- 解 已知 $x+y=4,则有 (x+y)^2 = x^2+2xy+y^2 = 16$
- 又已知 $x^2+y^2 = 10$,所以$(x+y)^2 = x^2+2xy+y^2 = 10+2xy = 16$
- 所以 $xy = (16-10)/2 = 3$

46. 若 $a + b = 6$，$a^2 + b^2 = 20$，求 $(a - b)^2$
- 解答 $a+b = 6$,则 $(a+b)^2 = a^2+2ab+b^2 = 36$,则$2ab = 36-20 = 16$
- $(a-b)^2 = a^2-2ab+b^2$,已知 $a^2+b^2 = 20,2ab=16$,因此$(a-b)^2 = 20-16 = 4$

47. 化简：$\frac{1}{x - 1} - \frac{1}{x + 1}$
- 解 $\frac{x+1}{x^2-1} - \frac{x-1}{x^2-1} = \frac{2}{x^2+1}$

48. 计算：$\frac{x^2 - 4}{x - 2} - \frac{x^2 - 1}{x + 1}$
- 解 $\frac{(x+2)(x-2)}{x-2} - \frac{(x+1)(x-1)}{x+1} = x+2 - (x- 1) = 3$ 且 $x\ne 2 或x\ne-1时成立$  

49. 已知 $p = x + \frac{2}{x}$，求 $x^2 + \frac{4}{x^2}$ 与 $p$ 的关系
- 解 $p^2 = (x+\frac{2}{x})^2 = x^2 + 2x\frac{2}{x} + \frac{2}{x^2} = x^2 + \frac{2}{x^2}$
- 因此$p^2 = x^2 + \frac{2}{x^2}$

50. 若 $(x + y)^2 = 25$，$(x - y)^2 = 9$，求 $xy$
- 解 $(x+y)^2 - (x-y) ^2 = x^2 + 2xy + y^2 - (x^2-2xy + y^2) = 4xy = 25 - 9 ,因此xy = 4$

### （一）基础方程（15道，浅→深）

1. 解方程：$3(x - 2) = 2x + 5$
- 解 $3x-6 -2x-5 = 0 => x-11 = 0,因此x=11$

2. 解方程：$\frac{x - 1}{2} - \frac{2x + 3}{3} = 1$
- $\frac{3x-3}{6} - \frac{4x+6}{6} = \frac{3x-3-4x-6}{6} = \frac{-x-9}{6}$

- $2x+2-3x+6 = 12=>-x+8 = 12=>x=-4$

4. 解方程：$5(x - 1) - 2(x + 3) = 4$
- $5x-5-2x-6 = 4 =>3x-11 = 4=>x=5$

5. 解一元二次方程：$x^2 - 4x + 3 = 0$
- $(x-1)(x-3) = 0=>x=1或x=3$

6. 解一元二次方程：$x^2 + 5x + 6 = 0$
- $(x+1)(x+5) = 0=>x=-1或x=-5$

7. 解一元二次方程：$2x^2 - 7x + 3 = 0$
- $(2x-1)(x-3) = 0=>x=\frac{1}{2}或者x=3$

9. 解一元二次方程：$2x^2 + 4x - 6 = 0$
- $(2x-2)(x+3) = 0=>x=1或者x=-3$

10. 解一元二次方程：$3x^2 - 2x - 1 = 0$
- $(3x+1)(x+1) = 0=>x=\frac{1}{3}或x=-1$

11. 已知方程 $x^2 - 2x + k = 0$ 有两个相等实根，求 $k$
- $x^2 - 2x + k = 0$ 有两个相等实根，则有 $b^2-4ac = -2^2-4k = 0,k = 1$

13. 已知方程 $x^2 - 5x + 6 = 0$ 的两根为 $x_1,x_2$，求 $x_1+x_2$ 与 $x_1x_2$
- $x_1 + x_2 = \frac{c}{a} = 6,x_1x_2 = -\frac{b}{a} = -\frac{5}{1} = 5$

14. 解方程：$(x - 2)(x + 3) = 5$
- 解 $x^2 + 3x - 2x - 6 = 5 => x^2 -x = 11$

15. 解分式方程：$\frac{2}{x} = \frac{3}{x + 1}$
- 解$2x+2=3x =>x=2$

33. 求二次函数 $y=x^2-4x+5$ 的顶点坐标和对称轴
