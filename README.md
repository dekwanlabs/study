# Study 项目目录说明

这个仓库已经按科目重新归类，当前建议从对应科目目录进入查看资料。

## 目录结构

- `计划/`
  - 放总学习计划、阶段安排、日程表

- `数学/`
  - 放考研数学补漏文档
  - 放高中数学教学版、题库、答案、练习题
  - 放数学相关示意图 `svg`

- `计算机408/`
  - 放考研计算机 `408` 专业课资料

- `英语二/`
  - 放考研英语二课程规划、单词与学习资料

## 使用建议

如果你是按当前备考节奏推进，建议优先这样看：

1. 先看 `计划/` 里的总计划
2. 数学进入 `数学/`
3. 专业课进入 `计算机408/`
4. 英语进入 `英语二/`

这样目录会比全部堆在根目录里清楚很多，后面继续补资料也更容易维护。
